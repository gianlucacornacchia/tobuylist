# Agent Instructions

- Before deploying the app to GitHub Pages, make sure all changes have been committed and pushed.
- you are running in a powershell environment, make sure to use the right syntax for console commands
- the app version has this format: vX.Y.Z 
X major revision
Y minor revision
Z patch revision
- for new feature increase the second number
- for minor changes increase the third number
- for bugfixes increase the third number
- for major changes increase the first number

## Documentation
- document each change in the CHANGELOG.md
- document each new feature in the README.md
- document each script in README.md
